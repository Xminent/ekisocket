Sends an HTTP Request to a server, specifying the HTTP method, URL, headers, and body. If you are using a specific HTTP method, you should consider using those instead of this function.

Parameters
methodThe HTTP Method to use.
urlThe URL to send the request to.
headersThe headers to send with the request.
bodyThe body of the request.
streamWhether or not to stream the response.
cbThe callback to call for each chunk of data received, if data is being streamed.
Returns
Response The response from the server.
